On Ubuntu: https://linoxide.com/linux-how-to/install-couchdb-futon-ubuntu-1604/

## Enable CORS in Remote CouchDB on ubuntu 16.04

### Install add-cors-to-couchdb using npm

```npm install -g add-cors-to-couchdb```

### Add CORS like so:
```add-cors-to-couchdb <you_url>:<port> -u <user_name> -p <your_password>```
